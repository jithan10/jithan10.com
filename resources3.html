🚀 Enable SSH on the Robot (Linux)

🔍 1. Check if SSH is Installed

which sshd

If nothing returns, you may need to install it. Otherwise, continue.

📦 2. Install SSH Server (if needed)

Only if sshd wasn't found in the previous step:

sudo apt update
sudo apt install openssh-server

⚙️ 3. Enable and Start SSH

sudo systemctl enable ssh
sudo systemctl start ssh

Check status:

sudo systemctl status ssh

You should see "active (running)".

🌐 4. Allow SSH Through Firewall (Optional)

sudo ufw allow ssh
sudo ufw enable
sudo ufw status

📡 5. Get the IP Address of the Robot

ip a

Look for the IP under wlan0 or eth0, usually something like 192.168.x.x

💻 6. Connect via SSH from Another Device

ssh root@192.168.x.x

If you're using a different user (like st):

ssh st@192.168.x.x
