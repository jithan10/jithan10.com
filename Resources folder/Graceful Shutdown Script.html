Graceful Shutdown Script on macOS

This guide walks you through setting up a custom terminal command called shutdown that gracefully closes all open applications on your Mac and then shuts it down. It's especially useful if you want to avoid force quitting apps or losing unsaved work.

1. Create the shutdown script

Open Terminal and run:

nano ~/shutdown

Then paste the following into the file:

#!/bin/bash

# Gracefully quit all user-facing apps
osascript <

2. Save and exit nano

  • Press Ctrl + O then Enter to save
  • Press Ctrl + X to exit

3. Make it executable

chmod +x ~/shutdown

4. Create an alias so you can use it like a command

echo 'alias shutdown="$HOME/shutdown"' >> ~/.zprofile

Then reload your terminal profile:

source ~/.zprofile

✅ All Set!

Now you can just type shutdown in your terminal and it will:

  1. Quit all open apps
  2. Wait 10 seconds
  3. Shutdown your Mac (you’ll be prompted for your password)
